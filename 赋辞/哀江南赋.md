# 哀江南赋

**作者：** 庾信  
**体裁：** 赋  

## 原文

粤以戊辰之年，建亥之月，金陵失险，石头不守。天子蒙尘于外，群公卿士，颠沛道路。余小子何德，兼年又稚，不预国谋，而遭家难。

于时东西南北，靡所可归。申包胥之顿地，碎之以首；蔡威公之泣血，尽之以目。钟仪君子，入就南冠之囚；季孙行人，留守西河之馆。

申胥谏而靡躯，子胥死而不悔。彼凌江而建德，或播迁于裔土。余之祖先，世济其美。桓桓祖武，绵绵世德。服冕乘轩，关石和钧。

及太清之季，皇家多故。天子避狄，播迁江表。钟山龙盘，石城虎踞。长江天堑，秦淮地险。而皇舆播越，终于覆败。

痛桓公之不作，悲管仲之无人。岂期终古，斯文遂绝！

于是江淮无主，河洛丘墟。生民涂炭，中原板荡。余乃窃叹曰："《春秋》之义，大一统也。昔者，圣人受命，奄有四海，统天下之大业，一六合之殊风。"

况复舟楫路穷，星汉非乘桴可越；风飙道阻，云山岂单车能通？

于是遂乃歔欷累欷，泪如雨下。心摧目断，魂飞魄散。

### 译文

在戊辰年十月，金陵失去险要，石头城不能坚守。天子在外蒙受风尘，群臣百官，在道路上颠沛流离。我这个小子有什么德行，年纪又幼稚，不参与国家大事的谋划，却遭遇家国的灾难。

在这时候东西南北，没有可以归依的地方。申包胥以头撞地，撞得头破血流；蔡威公哭泣流血，哭得眼睛都瞎了。钟仪这样的君子，进入牢狱成为戴南冠的囚徒；季孙这样的使者，留守在西河的馆舍。

申胥进谏而献身，子胥死了也不后悔。那些渡江建立功德的，有的流亡到边远的土地。我的祖先，世代继承美德。威武的祖先武功，连绵不断的世代德行。穿礼服坐高车，地位重要如关石和钧。

到了太清末年，皇室多灾多难。天子避难，流亡到江南。钟山像龙一样盘踞，石城像虎一样占据。长江是天然的壕沟，秦淮河是地理的险要。但是皇帝流亡，最终还是覆败了。

痛惜齐桓公不再出现，悲伤管仲这样的人才没有。哪里料到从此以后，这种文化就断绝了！

于是江淮没有主人，黄河洛水流域成为废墟。百姓陷入痛苦，中原动荡不安。我于是私下叹息说："《春秋》的大义，是天下大一统。从前，圣人受命，全部拥有四海，统一天下的大业，统一六合的不同风俗。"

况且船只道路断绝，银河不是坐竹筏可以渡过的；狂风道路阻塞，云山哪里是一辆车能够通过的？

于是就连续叹息，眼泪如雨下。心碎眼花，魂飞魄散。

## 赏析要点

### 文学特色

1. **抒情性强**：以强烈的主观情感贯穿全篇，情真意切
2. **叙事与抒情结合**：既有历史事件的叙述，又有深沉的情感抒发
3. **典故丰富**：大量运用历史典故，增强文章的文化内涵
4. **对比手法**：昔日繁华与今日衰败形成强烈对比

### 思想内容

1. **家国之痛**：表达了对梁朝灭亡、家国沦丧的深切痛苦
2. **历史感慨**：对历史兴衰的深刻思考和感慨
3. **文化忧虑**：对传统文化断绝的担忧和痛惜
4. **个人命运**：个人命运与国家命运紧密相连的悲剧意识

### 艺术成就

1. **情感真挚**：感情真挚深沉，具有强烈的感染力
2. **语言精美**：文辞华丽而不失自然，音韵和谐
3. **结构严谨**：层次分明，逻辑清晰，前后呼应
4. **意境深远**：营造出悲凉苍茫的艺术境界

### 历史背景

作品写于梁朝灭亡之后，庾信被迫留在北朝。作者以亲身经历，记录了梁朝末年的政治动乱和社会变迁，具有重要的史料价值。

### 文学影响

1. **开创先河**：开创了以个人经历反映历史变迁的赋体传统
2. **情感表达**：为后世抒情赋提供了典范
3. **史学价值**：具有重要的史学价值和文献价值
4. **艺术成就**：在赋体文学史上占有重要地位

### 与《枯树赋》的关系

《哀江南赋》与《枯树赋》都是庾信的代表作，两篇作品相互呼应：
- **主题相通**：都表达了对时光流逝、世事变迁的感慨
- **情感一致**：都体现了作者的家国之痛和个人悲剧
- **艺术风格**：都具有深沉的抒情色彩和丰富的文化内涵
- **历史价值**：都反映了南北朝时期的历史变迁

### 历史地位

《哀江南赋》是庾信的代表作，也是南北朝时期赋体文学的杰出作品。它以真挚的情感、精美的语言和深刻的思想内容，在中国文学史上占有重要地位，对后世文学创作产生了深远影响。
