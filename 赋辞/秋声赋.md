# 秋声赋
- 作者：欧阳修
- 体裁：赋

## 原文

欧阳子方夜读书，闻有声自西南来者，悚然而听之，曰："异哉！"初淅沥以萧飒，忽奔腾而砰湃，如波涛夜惊，风雨骤至。其触于物也，鏦鏦铮铮，金铁皆鸣；又如赴敌之兵，衔枚疾走，不闻号令，但闻人马之行声。

予谓童子："此何声也？汝出视之。"童子曰："星月皎洁，明河在天，四无人声，声在树间。"

予曰："噫嘻悲哉！此秋声也，胡为而来哉？盖夫秋之为状也：其色惨淡，烟霏云敛；其容清明，天高日晶；其气栗冽，砭人肌骨；其意萧条，山川寂寥。故其为声也，凄凄切切，呼号愤发。丰草绿缛，佳木葱茏，是皆夏之盛也，草拂之而色变，木遭之而叶脱。其所以摧败零落者，乃其一气之余烈。夫秋，刑官也，于时为阴；又兵象也，于行为金，是谓天地之义气，常以肃杀而为心。天之于物，春生秋杀，所以成岁功也。

吁嗟乎！草木无情，有时飘零。人为动物，惟物之灵；百忧感其心，万事劳其形；有动于中，必摇其精。而况思其力之所不及，忧其智之所不能，宜其渥然丹者为槁木，黟然黑者为星星。奈何以非金石之质，欲与草木而争荣？念谁为之戕贼，亦何恨乎秋声！"

童子莫对，垂头而睡。但闻四壁虫声唧唧，如助予之叹息。

## 注释

### 重要词汇注释

**欧阳子**：欧阳修自称。

**悚然**：惊惧的样子。

**异哉**：奇怪啊。

**淅沥**：象声词，形容雨声。

**萧飒**：象声词，形容风声。

**砰湃**：象声词，形容水声。

**鏦鏦铮铮**：象声词，形容金属撞击声。

**赴敌之兵**：奔赴战场的士兵。

**衔枚**：古代行军时士兵口中衔着的竹片，以防出声。

**明河**：银河。

**噫嘻**：叹词，表示感叹。

**胡为而来哉**：为什么来呢？

**盖夫**：大概，表示推测。

**惨淡**：暗淡。

**烟霏云敛**：烟雾缭绕，云彩收敛。

**天高日晶**：天空高远，日光明亮。

**栗冽**：寒冷。

**砭人肌骨**：刺透人的肌肤骨髓。

**萧条**：冷落，凄凉。

**寂寥**：空旷寂静。

**凄凄切切**：凄凉急切。

**呼号愤发**：呼啸愤怒地发出。

**绿缛**：绿色繁茂。

**葱茏**：草木茂盛。

**一气之余烈**：秋气的余威。

**刑官**：掌管刑罚的官员，这里指秋天主肃杀。

**于时为阴**：在时令上属于阴。

**兵象**：战争的象征。

**于行为金**：在五行中属金。

**义气**：正气。

**肃杀**：严厉杀伐。

**成岁功**：完成一年的功业。

**吁嗟乎**：叹词，表示感叹。

**动物**：有生命的东西。

**惟物之灵**：是万物中最有灵性的。

**百忧感其心**：各种忧愁感动其心。

**万事劳其形**：各种事务劳累其身。

**有动于中**：内心有所感动。

**必摇其精**：必然损伤其精神。

**思其力之所不及**：想到自己力量达不到的地方。

**忧其智之所不能**：担忧自己智慧不能解决的问题。

**渥然丹者**：红润的脸色。

**槁木**：枯木。

**黟然黑者**：乌黑的头发。

**星星**：稀疏，这里指白发。

**金石之质**：金石般坚硬的质地。

**争荣**：争夺荣华。

**戕贼**：伤害。

**莫对**：不回答。

**唧唧**：象声词，虫鸣声。

### 译文

欧阳修正在夜里读书，听到有声音从西南方向传来，惊恐地听着，说："奇怪啊！"开始像雨声淅沥萧瑟，忽然奔腾澎湃，如同波涛在夜里惊起，风雨突然到来。它碰到物体时，发出鏦鏦铮铮的声音，金属铁器都在鸣响；又像奔赴战场的士兵，口衔竹片急速行走，听不到号令声，只听到人马行进的声音。

我对童子说："这是什么声音？你出去看看。"童子说："星月明亮，银河在天，四周没有人声，声音在树林间。"

我说："唉，悲哀啊！这是秋声，为什么来了呢？大概秋天的形状是这样的：它的颜色暗淡，烟雾缭绕云彩收敛；它的容貌清明，天空高远日光明亮；它的气息寒冷，刺透人的肌肤骨髓；它的意境萧条，山川空旷寂静。所以它发出的声音，凄凉急切，呼啸愤怒。茂盛的绿草，繁茂的佳木，这些都是夏天的盛景，草被秋风吹拂就变色，树木遭遇秋风就落叶。它们之所以摧残败落，就是秋气的余威。秋天，是刑官，在时令上属于阴；又是战争的象征，在五行中属金，这就是所谓天地的正气，常常以严厉杀伐为本性。天对于万物，春天生长秋天凋零，这样来完成一年的功业。

唉！草木无情，有时也会飘零。人是有生命的动物，是万物中最有灵性的；各种忧愁感动其心，各种事务劳累其身；内心有所感动，必然损伤其精神。何况想到自己力量达不到的地方，担忧自己智慧不能解决的问题，应该是红润的脸色变成了枯木，乌黑的头发变成了稀疏的白发。为什么以非金石般坚硬的质地，要与草木争夺荣华呢？想想是谁伤害了自己，又何必怨恨秋声呢！"

童子不回答，垂着头睡着了。只听到四壁虫声唧唧，好像在帮助我叹息。

## 赏析

### 文学特色

1. **情景交融**：通过对秋声的描写，表达了作者对人生的感慨和哲理思考。

2. **象声词运用**：大量使用象声词，如"淅沥"、"萧飒"、"砰湃"、"鏦鏦铮铮"等，生动地表现了秋声的特点。

3. **对比手法**：将夏天的繁茂与秋天的萧瑟形成对比，突出秋天的肃杀之气。

4. **拟人手法**：将秋天比作刑官，赋予其人格化的特征。

### 思想内容

1. **人生感慨**：通过对秋声的感受，表达了对人生短暂、岁月无情的感慨。

2. **哲理思辨**：从自然现象上升到人生哲理，体现了作者深刻的思想。

3. **时光意识**：强调时间的流逝和生命的有限，体现了古代文人的时间观念。

4. **自然观念**：认为自然有其规律，春生秋杀是天地的常理。

### 艺术成就

1. **结构完整**：从听声、问声、释声到感声，结构严谨，层次分明。

2. **语言优美**：文辞华美，音韵和谐，体现了散文的艺术美。

3. **意境深远**：通过具体的声音描写，营造出深远的意境。

4. **情理并茂**：既有感情的抒发，又有理性的思辨，达到了情理并茂的效果。

### 在文学史上的地位

1. **宋代散文代表**：是宋代散文的代表作品，体现了宋代散文的特色。

2. **咏物赋典范**：在咏物赋中具有典范意义，影响了后代的创作。

3. **哲理性强**：体现了宋代文学注重哲理思辨的特点。

4. **艺术价值高**：在艺术表现上达到了很高的水平，被誉为名篇。

### 与其他秋赋的比较

1. **与《秋兴赋》比较**：潘岳的《秋兴赋》更多地表现个人的悲秋情怀，而欧阳修的《秋声赋》更注重哲理思辨。

2. **与《悲秋赋》比较**：相比于一般的悲秋作品，《秋声赋》在感情表达上更加含蓄深沉。

3. **独特之处**：通过"声"这一独特角度来写秋，别具一格，开创了新的表现手法。
