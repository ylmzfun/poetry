四书五经
========

四书五经是四书、五经的合称，泛指儒家经典著作。
四书指的是《大学》《中庸》《论语》《孟子》。五经指的是《诗经》《尚书》《礼记》《周易》《春秋》五部。《礼记》通常包括三礼，即《仪礼》《周礼》《礼记》；《春秋》由于文字过于简略，通常与解释《春秋》的《左传》《公羊传》《谷梁传》分别合刊。四书之名始于宋朝，五经之名始于汉武帝。 来自百科.
