# 步出夏门行 土不同

**作者：** 佚名  
**朝代：** 魏晋南北朝  
**体裁：** 四言诗  
**出处：** 《全宋词》

## 原文

乡土不同，河朔隆冬。
流澌浮漂，舟船行难。
锥不入地，蘴藾深奥。
水竭不流，冰坚可蹈。
士隐者贫，勇侠轻非。
心常叹怨，戚戚多悲。
幸甚至哉！歌以咏志。
