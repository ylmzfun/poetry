# 善哉行 其三

**作者：** 佚名  
**朝代：** 魏晋南北朝

## 原文

朝日乐相乐，酣饮不知醉。
悲弦激新声，长笛吹清气。
弦歌感人肠，四坐皆欢悦。
寥寥高堂上，凉风入我室。
持满如不盈，有德者能卒。
君子多苦心，所愁不但一。
慊慊下白屋，吐握不可失。
众宾饱满归，主人苦不悉。
比翼翔云汉，罗者安所羁？
冲静得自然，荣华何足为！
