# 兵要望江南 占雨第八（京本列第二○十七首） 七

**作者：** 易靜  
**朝代：** 唐

## 原文

兵（首本作「風」，此從辛本）發日，風雨逆沾人。
天意示（辛本作「欲」）知教我記，不須前進恐危身，見陣潰亡軍。
（《軍行志》：如行軍見暴風猛雨，倒折旗鎗，大雨不止者，天之怒也，軍不可進。
宜待天象和明，別擇日時門戶，軍行方吉也。
）。
