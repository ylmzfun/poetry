# 觀楚州韋舍人新築河堤怺建兩閶門（按此詩亦見《永樂大典》卷三五二七「九真」「門」字「北閶門」條〖五函四十九冊〗。《大典》引《元一統志》曰：「淮安路有北閶門，唐〖張，原脫〗祐〖應作祜〗有《陪楚州韋舍人北閶門遊燕詩》。」）

## 原文

宴賓容易小筵成，隼擊秋原助放情。
紅袖退行鞍上語，白眉迂步馬前迎。
一冬霜意先黃葉，兩路風威動翠旌。
須道孔融罇已滿，不勞臺下說餘酲。
