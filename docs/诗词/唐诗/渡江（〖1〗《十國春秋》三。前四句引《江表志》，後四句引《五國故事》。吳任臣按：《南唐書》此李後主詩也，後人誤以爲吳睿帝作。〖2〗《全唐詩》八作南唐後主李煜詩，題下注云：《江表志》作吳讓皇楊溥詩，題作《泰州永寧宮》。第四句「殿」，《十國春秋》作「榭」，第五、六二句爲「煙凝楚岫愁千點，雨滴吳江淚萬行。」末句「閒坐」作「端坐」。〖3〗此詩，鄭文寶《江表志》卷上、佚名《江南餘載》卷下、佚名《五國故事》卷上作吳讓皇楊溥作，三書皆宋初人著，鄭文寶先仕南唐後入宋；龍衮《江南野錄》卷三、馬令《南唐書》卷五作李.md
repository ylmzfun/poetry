# 渡江（〖1〗《十國春秋》三。前四句引《江表志》，後四句引《五國故事》。吳任臣按：《南唐書》此李後主詩也，後人誤以爲吳睿帝作。〖2〗《全唐詩》八作南唐後主李煜詩，題下注云：《江表志》作吳讓皇楊溥詩，題作《泰州永寧宮》。第四句「殿」，《十國春秋》作「榭」，第五、六二句爲「煙凝楚岫愁千點，雨滴吳江淚萬行。」末句「閒坐」作「端坐」。〖3〗此詩，鄭文寶《江表志》卷上、佚名《江南餘載》卷下、佚名《五國故事》卷上作吳讓皇楊溥作，三書皆宋初人著，鄭文寶先仕南唐後入宋；龍衮《江南野錄》卷三、馬令《南唐書》卷五作李

## 简介

《渡江（〖1〗《十國春秋》三。前四句引《江表志》，後四句引《五國故事》。吳任臣按：《南唐書》此李後主詩也，後人誤以爲吳睿帝作。〖2〗《全唐詩》八作南唐後主李煜詩，題下注云：《江表志》作吳讓皇楊溥詩，題作《泰州永寧宮》。第四句「殿」，《十國春秋》作「榭」，第五、六二句爲「煙凝楚岫愁千點，雨滴吳江淚萬行。」末句「閒坐」作「端坐」。〖3〗此詩，鄭文寶《江表志》卷上、佚名《江南餘載》卷下、佚名《五國故事》卷上作吳讓皇楊溥作，三書皆宋初人著，鄭文寶先仕南唐後入宋；龍衮《江南野錄》卷三、馬令《南唐書》卷五作李》是中国古代文学史上的重要作品，体现了当时的文学风格和思想情感。这首作品具有较高的文学价值和历史意义。
