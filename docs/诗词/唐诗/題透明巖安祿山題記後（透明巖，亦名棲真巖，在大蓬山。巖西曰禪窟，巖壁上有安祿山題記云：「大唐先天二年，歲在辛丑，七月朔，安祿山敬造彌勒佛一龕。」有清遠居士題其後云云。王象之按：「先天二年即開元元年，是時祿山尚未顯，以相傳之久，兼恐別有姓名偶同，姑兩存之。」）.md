# 題透明巖安祿山題記後（透明巖，亦名棲真巖，在大蓬山。巖西曰禪窟，巖壁上有安祿山題記云：「大唐先天二年，歲在辛丑，七月朔，安祿山敬造彌勒佛一龕。」有清遠居士題其後云云。王象之按：「先天二年即開元元年，是時祿山尚未顯，以相傳之久，兼恐別有姓名偶同，姑兩存之。」）

## 原文

妖胡作逆罪滔天，翠輦倉皇幸蜀川。
千載業緣磨不盡，卻來邀福向金仙。
（《輿地紀勝》一八八《蓬州·景物》及《碑記》）（〖1〗葉奕苞《金石錄補》卷十二云：「居士不知何許人。
顏魯公于虎邱題詠，同游者亦曰清遠，未知即其人否。
」）。
