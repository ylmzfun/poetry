# 句（度，工詩，有句云云，人多誦之。王朴爲樞密，止以此一聯薦於申文炳知舉，遂擢爲第三。人嘲曰：「主司只誦一聯詩。」）

**作者：** 李度  
**朝代：** 唐

（《玉壺清話》七）。
