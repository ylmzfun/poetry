# 歌（二章）（高祖行密改元開國時，廣陵殷盛，士庶駢闐。道人狀如病狂，手持一竿，竿頭懸一木，刻爲鯉魚形，行歌於市云云。其類此意者，凡數十章，時人莫能曉。後徐知誥禪代，復姓李氏，其言始應。） 二

## 原文

橫排三十六條鱗，個個圓如紫磨真。
爲甚竿頭挑着定，世間難遇識魚人。
（《十國春秋·吳》十二。
又《全五代詩》二十三轉引）（〖1〗見《釣磯立談》）。
