# 安西館中思長安

**作者：** 岑參  
**朝代：** 唐  
**体裁：** 古体诗  
**出处：** 《全唐诗》

## 原文

家在日出處，朝來起東風。風從帝鄉來，不異家信通。
絕域地欲盡，孤城天遂窮。彌年但走馬，終日隨飄蓬。
寂寞不得意，辛勤方在公。胡塵淨古塞，兵氣屯邊空。
鄉路眇天外，歸期如夢中。遙憑長房術，為縮天山東。
