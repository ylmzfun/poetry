# 春台引

**作者：** 陳子昂  
**朝代：** 唐  
**体裁：** 楚辞  
**出处：** 《全唐诗》

## 原文

感陽春兮生碧草之油油。懷宇宙以傷遠，登高台而寫憂。
遲美人兮不見，恐青歲之遂遒。從畢公以酣飲，
何雲木之美麗，而池館之崇幽。星台秀士，月旦諸子。
嘉青鳥之辰，迎火龍之始。挾寶書與瑤瑟，
擊青鐘，歌淥水。怨青春之萎絕，贈瑤台之旖旎。
願一見而道意，結眾芳之綢繆。曷餘情之蕩漾，
群仙去兮青春頹，歲華歇兮黃鳥哀。富貴榮樂幾時兮，
