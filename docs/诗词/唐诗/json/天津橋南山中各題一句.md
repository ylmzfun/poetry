# 天津橋南山中各題一句

**作者：** 李益  
**朝代：** 唐  
**体裁：** 五言诗  
**出处：** 《全唐诗》

## 原文

野坐分苔席，——李益
山行繞菊叢。——韋執中
雲衣惹不破，——諸葛覺
秋色望來空。——賈島
