# 人事

**作者：** 徐夤  
**朝代：** 唐  
**体裁：** 古体诗  
**出处：** 《全唐诗》

## 原文

人事飄如一炷煙，且須求佛與求仙。豐年甲子春無雨，
平生生計何為者，三徑蒼苔十畝田。
