# 大蜀皇帝壽春節進堯銘舜頌二首：堯銘

**作者：** 貫休  
**朝代：** 唐  
**体裁：** 古体诗

## 原文

金冊昭昭，列聖孤標。仲尼有言，巍巍帝堯。承天眷命，
吾皇則之，小心翼翼。秉陽亭毒，不遑暇食。土階苔綠，
漢高將將，太宗兵柄。吾皇則之，日新德盛。朽索六馬，
既玉其葉，亦金其枝。葉葉枝枝，百工允厘。享國如堯，
