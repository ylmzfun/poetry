# 病後遇王倚飲，贈歌

**作者：** 杜甫  
**朝代：** 唐  
**体裁：** 古体诗

## 原文

麟角鳳觜世莫識，煎膠續弦奇自見。尚看王生抱此懷，
酷見凍餒不足恥，多病沈年苦無健。王生怪我顏色惡，
頭白眼暗坐有胝，肉黃皮皺命如線。惟生哀我未平復，
長安冬菹酸且綠，金城土酥靜如練。兼求富豪且割鮮，
