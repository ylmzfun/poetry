# 西江月（六首。以下詞）（原八首，前二首已見《全唐詩》，此六首，《全宋詞》亦未收。） 三

## 原文

聽說金公兩字，何物喚作金孫。
尋枝尋葉必知根，無智便乃心昏。
若用凡鉛爲體，都來少魄無魂。
水銀漸結必難存，祕訣要處誰論。
