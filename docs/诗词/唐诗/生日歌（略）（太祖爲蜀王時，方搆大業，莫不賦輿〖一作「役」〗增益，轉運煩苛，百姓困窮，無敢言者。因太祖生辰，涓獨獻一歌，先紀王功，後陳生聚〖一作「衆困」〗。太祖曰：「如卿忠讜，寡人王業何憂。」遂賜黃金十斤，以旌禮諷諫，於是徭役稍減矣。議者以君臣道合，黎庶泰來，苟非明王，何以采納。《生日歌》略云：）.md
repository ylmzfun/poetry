# 生日歌（略）（太祖爲蜀王時，方搆大業，莫不賦輿〖一作「役」〗增益，轉運煩苛，百姓困窮，無敢言者。因太祖生辰，涓獨獻一歌，先紀王功，後陳生聚〖一作「衆困」〗。太祖曰：「如卿忠讜，寡人王業何憂。」遂賜黃金十斤，以旌禮諷諫，於是徭役稍減矣。議者以君臣道合，黎庶泰來，苟非明王，何以采納。《生日歌》略云：）

## 原文

百姓富，軍食足，百姓足，軍民歡。
爭那生靈饑且寒，吾王有術應不難。
但令一斗徵一斗，自然百姓富於官。
