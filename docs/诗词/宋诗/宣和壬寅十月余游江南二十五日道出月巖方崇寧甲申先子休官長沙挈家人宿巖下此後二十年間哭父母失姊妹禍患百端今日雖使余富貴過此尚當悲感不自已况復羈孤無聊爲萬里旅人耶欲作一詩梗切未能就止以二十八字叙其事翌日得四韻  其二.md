# 宣和壬寅十月余游江南二十五日道出月巖方崇寧甲申先子休官長沙挈家人宿巖下此後二十年間哭父母失姊妹禍患百端今日雖使余富貴過此尚當悲感不自已况復羈孤無聊爲萬里旅人耶欲作一詩梗切未能就止以二十八字叙其事翌日得四韻  其二

## 原文

石漏搖空一片天，月巖之號古相傳。
孤輪高潔誰爲比，老樹婆娑亦宛然。
山礙不容千里見，崖侵常蝕二分偏。
故知僞物誰叨冒，終竟天教不十全。
