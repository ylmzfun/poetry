# 海外氣候每歲三四月間已如劇暑客有自吉陽至者寓館問漢亭累日且言吉陽氣候晝夜如炊因歎此邦之勝乃知人生無有足時不經熱惱豈知平日之清凉乎故古之達者毎以此對治釋氏云推落大火坑火坑變成池皮鞋和尚以爲即時清凉也蘇公亦云嶺南瘴毒地有此江月寒乃知天壤間何人不清安予謫居嶺海踰十五年見聞習熟不以爲異因作此詩以自慰且以警世之賤丈夫一不快即愁歎怨憤或譏謗怒罵如柳劉之徒蓋未足以語此也

## 简介

《海外氣候每歲三四月間已如劇暑客有自吉陽至者寓館問漢亭累日且言吉陽氣候晝夜如炊因歎此邦之勝乃知人生無有足時不經熱惱豈知平日之清凉乎故古之達者毎以此對治釋氏云推落大火坑火坑變成池皮鞋和尚以爲即時清凉也蘇公亦云嶺南瘴毒地有此江月寒乃知天壤間何人不清安予謫居嶺海踰十五年見聞習熟不以爲異因作此詩以自慰且以警世之賤丈夫一不快即愁歎怨憤或譏謗怒罵如柳劉之徒蓋未足以語此也》是中国古代文学史上的重要作品，体现了当时的文学风格和思想情感。这首作品具有较高的文学价值和历史意义。

## 原文

客自東來說吉陽，始知儋耳本清凉。
潮聲卷海千峰雨，月色侵門滿地霜。
更欲何方尋佛國，此生真欲老蠻鄉。
安心守一師吾祖，尚覺人間日月長。
