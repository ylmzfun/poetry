# 東坡先生與蔣魏公遊最善宣卿侍郎蓄東坡詩文自公始也心慕手追遂入於室某嘗賦景坡堂詩宣卿謂余知音者遂標藏之櫝中比守吳門治有狀璽書褒進待制敷文閣某馳小舟往賀宣卿出詩三章見屬句法華妙爲一時絕唱有云正索解人那復得其誰知我固無從此真東坡語也輒次韻書于卷末  其三

## 原文

洲蘂霏紅風澹沱，池光鋪練月徘徊。
乍驚醉眼生春纈，頓覺詩腸蟄夜雷。
坐看千言供刻燭，自甘百罰恥空罍。
使君處處栽桃李，爲報花奴羯鼓催。
