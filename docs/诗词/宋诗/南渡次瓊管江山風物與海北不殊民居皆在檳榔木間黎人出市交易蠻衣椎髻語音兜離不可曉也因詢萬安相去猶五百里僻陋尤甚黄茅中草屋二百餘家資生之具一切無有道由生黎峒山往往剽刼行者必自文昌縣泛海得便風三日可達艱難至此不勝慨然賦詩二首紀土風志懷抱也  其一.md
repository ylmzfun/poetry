# 南渡次瓊管江山風物與海北不殊民居皆在檳榔木間黎人出市交易蠻衣椎髻語音兜離不可曉也因詢萬安相去猶五百里僻陋尤甚黄茅中草屋二百餘家資生之具一切無有道由生黎峒山往往剽刼行者必自文昌縣泛海得便風三日可達艱難至此不勝慨然賦詩二首紀土風志懷抱也  其一

**朝代：** 宋

## 原文

巨舶浮于海，長飈送短蓬。
夜潮和月白，曉日跳波紅。
雲影搖脩浪，瀾光接遠空。
喜過三河流，愁遠冠頭峰。
雷化迷天際，瓊儋入望中。
地遥横一線，山露點羣鴻。
偶脫鯨鯢患，尤欣氣俗同。
川原驚老眼，稚耋看衰翁。
蠻市蝦魚合，賓居棟宇雄。
人煙未寥落，竹樹自葱蘢。
碧暗檳榔葉，香移薄荷叢。
金花翔孔翠，綵幕問黎童。
南極冬猶暖，中原信不通。
管寧雖跡遠，阮籍已途窮。
澒洞滄波裏，蒼茫返照東。
客愁渾不寢，鼓角五更風。
