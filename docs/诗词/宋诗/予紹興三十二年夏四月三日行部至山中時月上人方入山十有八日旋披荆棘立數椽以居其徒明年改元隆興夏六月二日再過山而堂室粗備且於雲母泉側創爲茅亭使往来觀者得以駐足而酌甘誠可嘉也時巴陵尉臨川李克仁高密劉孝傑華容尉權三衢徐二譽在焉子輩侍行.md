# 予紹興三十二年夏四月三日行部至山中時月上人方入山十有八日旋披荆棘立數椽以居其徒明年改元隆興夏六月二日再過山而堂室粗備且於雲母泉側創爲茅亭使往来觀者得以駐足而酌甘誠可嘉也時巴陵尉臨川李克仁高密劉孝傑華容尉權三衢徐二譽在焉子輩侍行

## 原文

去年入山時，孟夏暑猶薄。
来觀雲母泉，陰崖坐盤礴。
老禪披荆榛，經營始興作。
山空斤斧響，丁丁振林壑。
俛仰恰周星，炎赫若凌薄。
昨朝發岳陽，東風怒還惡。
驚濤送輕舟，十里沙頭落。
投凉不待曉，肩輿上磽确。
僧徒笑相迎，此意安可卻。
高亭蔭白茅，群峰峙青廓。
偕行二三子，臨泉一杯酌。
呼僮瀹石鼎，煮湯旋和藥。
誰令俗士駕，急去能不怍。
因成數句語，聊發後来噱。
