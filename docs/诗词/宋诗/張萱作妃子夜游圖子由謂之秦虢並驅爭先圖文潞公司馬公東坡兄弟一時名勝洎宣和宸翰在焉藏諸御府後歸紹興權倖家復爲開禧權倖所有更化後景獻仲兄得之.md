# 張萱作妃子夜游圖子由謂之秦虢並驅爭先圖文潞公司馬公東坡兄弟一時名勝洎宣和宸翰在焉藏諸御府後歸紹興權倖家復爲開禧權倖所有更化後景獻仲兄得之

## 原文

孰揮若木枝，日馭不可勒。
志士分寸惜，長繩繫不得。
忽觀此圖使我傷，使我痛憤嗟三郎。
三郎飛入月中去，回首不知宮漏長。
金羈金勒控紫韁，玉花玉面下采床。
天階夜色浄如洗，宮彎促𩍐相抵當。
十家三國歸模冩，不冩漁陽閙胡馬。
較來不畫亦不妨，冏冏莫掩張曲江。
宣和宸奎獨爛爛，元祐名卿題得滿。
既歸天上誇襲藏，又落人間作奇玩。
沉思惡衣菲飲食，過門不子呱呱泣。
張萱已矣倘可呼，爲我只作塗山圖。
