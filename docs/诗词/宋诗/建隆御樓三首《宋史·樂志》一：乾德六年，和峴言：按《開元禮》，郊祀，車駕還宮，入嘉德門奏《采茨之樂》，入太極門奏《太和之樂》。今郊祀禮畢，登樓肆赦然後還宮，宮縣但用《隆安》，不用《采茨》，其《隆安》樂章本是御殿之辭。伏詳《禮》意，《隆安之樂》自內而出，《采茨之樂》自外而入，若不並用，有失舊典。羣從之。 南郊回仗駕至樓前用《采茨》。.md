# 建隆御樓三首《宋史·樂志》一：乾德六年，和峴言：按《開元禮》，郊祀，車駕還宮，入嘉德門奏《采茨之樂》，入太極門奏《太和之樂》。今郊祀禮畢，登樓肆赦然後還宮，宮縣但用《隆安》，不用《采茨》，其《隆安》樂章本是御殿之辭。伏詳《禮》意，《隆安之樂》自內而出，《采茨之樂》自外而入，若不並用，有失舊典。羣從之。 南郊回仗駕至樓前用《采茨》。

## 原文

高煙升太一，明祀達乾坤。
天仗回嶢闕，皇輿入應門。
簪裳如霧集，車騎若雲屯。
兆庶皆翹首，巍巍萬乘尊。
