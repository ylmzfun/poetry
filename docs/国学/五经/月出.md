# 月出

**作者：** 佚名  
**朝代：** 未知  
**体裁：** 楚辞  
**出处：** 《陈风》

## 原文

月出皎兮。佼人僚兮。舒窈纠兮。劳心悄兮。
月出皓兮。佼人懰兮。舒忧受兮。劳心慅兮。
月出照兮。佼人燎兮。舒夭绍兮。劳心惨兮。
