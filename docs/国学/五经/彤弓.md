# 彤弓

**作者：** 佚名  
**朝代：** 未知  
**体裁：** 楚辞  
**出处：** 《南有嘉鱼之什》

## 原文

彤弓弨兮，受言藏之。我有嘉宾，中心贶之。钟鼓既设，一朝飨之。
彤弓弨兮，受言载之。我有嘉宾，中心喜之。钟鼓既设，一朝右之。
彤弓弨兮，受言櫜之。我有嘉宾，中心好之。钟鼓既设，一朝酬之。
