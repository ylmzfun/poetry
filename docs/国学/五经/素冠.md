# 素冠

**作者：** 佚名  
**朝代：** 未知  
**体裁：** 楚辞  
**出处：** 《桧风》

## 原文

庶见素冠兮，棘人栾栾兮。劳心慱慱兮。
庶见素衣兮，我心伤悲兮。聊与子同归兮。
庶见素韠兮，我心蕴结兮。聊与子如一兮。
