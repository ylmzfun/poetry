# 鸿雁

**朝代：** 未知

## 原文

鸿雁于飞，肃肃其羽。之子于征，劬劳于野。爰及矜人，哀此鳏寡。
鸿雁于飞，集于中泽。之子于垣，百堵皆作。虽则劬劳，其究安宅？
鸿雁于飞，哀鸣嗷嗷。维此哲人，谓我劬劳。维彼愚人，谓我宣骄。
