# 蓼萧

**作者：** 佚名  
**朝代：** 未知  
**体裁：** 楚辞  
**出处：** 《南有嘉鱼之什》

## 原文

蓼彼萧斯，零露湑兮。既见君子，我心写兮。燕笑语兮，是以有誉处兮。
蓼彼萧斯，零露瀼瀼。既见君子，为龙为光。其德不爽，寿考不忘。
蓼彼萧斯，零露泥泥。既见君子，孔燕岂弟。宜兄宜弟，令德寿岂。
蓼彼萧斯，零露浓浓。既见君子，鞗革忡忡。和鸾雍雍，万福攸同。
