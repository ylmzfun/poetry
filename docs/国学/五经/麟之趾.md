# 麟之趾

**作者：** 佚名  
**朝代：** 未知  
**体裁：** 楚辞  
**出处：** 《周南》

## 原文

麟之趾，振振公子，于嗟麟兮。
麟之定，振振公姓，于嗟麟兮。
麟之角，振振公族，于嗟麟兮。
